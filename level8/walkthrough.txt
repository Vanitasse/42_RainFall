what does these commands do ?

auth calls malloc() and strcpy() and copies datas following auth into the global auth.

service calls strdup() and copies  datas following service into the global service.

reset calls free() and fress the global auth.

login calls system() if auth(32) is not 0, or fwrite() otherwise. then displays addresses of the globals auth and service.

by doing one auth and one service, we can see that the address of service is 16 bytes after the adress of auth. this is because malloc puts each pointer next to the other one with some padding.
